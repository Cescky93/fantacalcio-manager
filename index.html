    Archivio giornate

    Come usare Fantacalcio Rosa Live

    1. Carica la rosa

    Vai in Dati → Importa rosa rapida e incolla una lista CSV/testo, oppure importa un file CSV. Puoi anche aggiungere i giocatori uno alla volta dalla sezione Rosa.

    2. Aggiorna la giornata

    In Giornata imposta numero giornata, modulo preferito, scadenza formazione e note personali.

    3. Controlla le news

    In News Live incolla probabili formazioni, indisponibili o appunti presi da fonti esterne. L'app cerca i tuoi giocatori e propone status aggiornati.

    4. Genera la formazione

    In Formazione premi “Suggerisci dai dati live”. Poi correggi manualmente se hai sensazioni diverse.

    5. Usa IA gratis

    In Analisi IA scegli Fast/Standard/Deep, copia la richiesta blindata e incollala in ChatGPT, Gemini o Claude. La richiesta è ancorata alla giornata Serie A indicata.

    6. Backup

    In Dati esporta JSON per salvare la rosa o passarla a un altro dispositivo. Ogni telefono mantiene i propri dati separati.

    Formato consigliato per import veloce

    Il formato più sicuro è una riga per giocatore, separata da punto e virgola:

    Ruolo;Nome;Squadra;Costo;Quotazione;Note
P;Sommer;Inter;12;18;Portiere titolare
D;Dimarco;Inter;42;25;Top difesa
C;Orsolini;Bologna;32;24;Bonus
A;Lautaro Martinez;Inter;105;40;Top attacco

    Va bene anche un CSV esportato dall'app ufficiale, purché contenga almeno nome e ruolo. Se l'esportazione ha colonne diverse, l'app prova a riconoscerle automaticamente.

    Impostazioni

    Importa rosa rapida

    Incolla una lista manuale o CSV. Formato consigliato: Ruolo;Nome;Squadra;Costo;Quotazione;Note. Puoi anche caricare un file CSV esportato dall'app ufficiale.

    Backup

    I dati restano sul dispositivo. Per passare la rosa a un altro telefono usa esporta/importa JSON.
